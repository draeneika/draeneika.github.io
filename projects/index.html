Zadanie 1: Osobná webová prezentácia na GitHub pages

  • Zadanie:
    Vytvorte webovú prezentáciu (webové sídlo) o sebe. Zamerajte sa jednak na vaše profesné záujmy (napr. projekty, ktoré riešite/riešili ste, čo vás v informatike najviac baví, fascinuje = váš developerský profil) a jednak vaše osobné záujmy, hobby. V rámci developerského profilu vytvorte sekciu Webové publikovanie, kde budete publikovať všetky tri vaše vypracované zadania z predmetu. Využite pritom technológie Git + GitHub Pages + Jekyll + Markdown. Využite potenciál statického generátora Jekyll a jeho templatovacích možností.

  • Podrobné požiadavky na vypracovanie a odovzdanie zadania (priemerná úroveň kvality):
    • Sídlo musí obsahovať aspoň 5 podstránok, pri využití aspoň 3 rôznych rozložení (layout-ov)
    • V rámci šablon musí byť použité:
      • aspoň 5 premenných
      • kolekcie alebo dátové súbory
      • aspoň 5 filtrov alebo tagov
      • aspoň 1 plugin (okrem pagination)

  • Dokumentácia
    • 5 podstránok:
      1. Home
      2. About
      3. Education
      4. Jobs & Skills
      5. Favorite Webtoons
      6. Webové publikovanie
    • 3 layouts:
      1. default
      2. right_photo
      3. cards
    • 5 premenných:
      1. skills + atribúty
      2. jobs + atribúty
      3. webtoons + atribúty
      4. shools + atribúty
      5. sorted_skills
    • kolekcie:
      1. _schools
      2. _jobs
      3. _skills
      4. _webtoons
    • plugin:
      • jemoji
    • taspoň 5 filtrov alebo tagov:
      • tagy:
        1. {% raw %} {% for job in site.jobs %} {% endraw %}
        2. {% raw %} {% include navigation.html %} {% endraw %}
        3. {% raw %} {% assign sorted_skills = site.skills | sort: "name" %} {% endraw %} - vytvorenie novej premennej
        4. raw - na escaping
      • filtre:
        1. {% raw %} {{job.from | date: "%B %Y" }} {% endraw %} - na zobrazenie dátumu v tvare Mesiac Rok
        2. {% raw %}{% assign sorted_skills = site.skills | sort: "name" %} {% endraw %} - zoradenie poľa abecedne podla atribútu name
